# 交易層級（Trading Timeframe Levels）知識庫

> 此文件用途：作為交易策略分析工具的背景知識 context，可直接貼入 Claude Code 的 system prompt 或 user prompt。

---

## 核心概念定義

### 什麼是層級（Timeframe Level）

K 線依時間週期分為不同「層級」，常見由大到小如下：

```
月K > 週K > 日K > 60分K > 30分K > 10分K > 5分K > 1分K
```

- 週 K = 5 根日 K 的集合
- 30 分 K = 30 根 1 分 K 的集合
- **小層級的走勢最終會匯總成大層級**，大層級趨勢尚未結束，小層級通常也不會快速結束

### 強弱判斷原則

同一根 K 線的突破，在不同大層級下意義完全不同：

- 大層級（週K）多頭排列 + 小層級也多頭排列 → **強勢買點**
- 大層級正在修正下跌 → 小層級突破可能只是短暫反彈，進場容易套牢

## 核心策略：看大做小（Top-Down Analysis）

### 策略邏輯

| 角色 | 時間週期範例 | 功能 |
|------|-------------|------|
| 大層級（主趨勢） | 週K / 日K | 確認方向、過濾雜訊 |
| 小層級（執行層） | 60分K / 5分K | 找進場點、設停損 |

**順勢定義**：當小層級走勢方向與大層級趨勢一致時，才算真正順勢。

### 操作流程

```
步驟 1：確認大層級趨勢方向（做多/做空/觀望）
步驟 2：等待大層級出現「停頓」訊號
步驟 3：切換到小層級尋找精確進場點
步驟 4：在小層級設定停損點
步驟 5：以大層級趨勢作為持倉依據
```

### 停頓訊號的特徵（切換小層級的時機）

大層級出現以下 K 線特徵時，切入小層級找買點：

- 非長紅 K（上漲幅度縮小）
- 小紅 K / 小黑 K
- 振幅小、不見高不破低的 K 線
- 暫時止漲的跡象

## 層級比例（Ratio）選擇原則

### 合適比例範例

```
日K（主）    + 60分K（執行）  ✅ 推薦
60分K（主）  + 5分K（執行）   ✅ 推薦
週K（主）    + 日K（執行）    ✅ 推薦
```

### 錯誤比例

```
日K（主）    + 1分K（執行）   ❌ 比例過大 → 1分K雜訊太多，反覆停損
日K（主）    + 4小時K（執行） ❌ 比例過小 → 無法有效區分短中期趨勢
```

### 層級數量

- 一般使用 **2 個層級** 即可
- 不是層級越多成功率越高，過多層級反而增加決策複雜度
- 有深厚基礎後才考慮加入第 3 個層級

## 找強勢行情的方法

### 最強訊號：大小層級均線多排

```
條件 1：週K → 均線多排（5MA > 10MA > 20MA > 55MA 等）
條件 2：切入日K / 小層級 → 確認也是均線多排
條件 3：兩層級都確認 → 判定為強勢行情
動作：在小層級找買點、設停損
```

### 適用範圍

此邏輯通用於：**台股、台指期、外匯、加密貨幣** 等任何有 K 線圖且流動性足夠的標的。

## 進場與停損邏輯

### 進場點本質

「買點」= **大層級的轉折位置，同時也是小層級的突破位置**

```
不要追：大層級（主層級）的突破
可以追：小層級（執行層級）的突破

原因：小層級突破 → 往往對應大層級出現長紅K、一過二、5MA轉折等買點
```

做多/做空對應關係：

| 方向 | 追的是 | 大層級對應 |
|------|--------|-----------|
| 做多 | 小層級突破（向上） | 長紅K / 一過二 / 5MA上揚 |
| 做空 | 小層級跌破（向下） | 長黑K / 一破二 / 5MA下彎 |

### 小層級停損的優勢

- 停損點更精確，損失更小
- 降低持股平均成本 → 提高持倉信心 → 增加持倉時間 → 提升獲利機率

## 層級選擇指南（依交易者狀況）

| 條件 | 建議層級 |
|------|---------|
| 風險承受高、資金大 | 週K / 日K（趨勢穩定，訊號少）|
| 風險承受低、資金小 | 5分K / 1分K（停損小，但停損次數多）|
| 初學者 | 小層級 + 極少資金（練習用）|
| 有一定經驗 | 30分K / 60分K |
| 進階交易者 | 日K 配 60分K，或依市場波動彈性調整 |

> ⚠️ 重要原則：選擇的層級停損金額，必須是自己可以承受的。沒有最好的搭配，只有最適合自己的。

## 層級切換時機整理

### 大切小（找進場點）

**觸發條件**：大層級出現停頓 K 線
**動作**：切換到小層級，找精確買入/賣出點

### 小切大（確認趨勢）

**觸發條件**：小層級出現橫盤整理、訊號模糊、看不懂行情
**動作**：切換到大層級，確認整體波段走勢方向，這也是能否抱緊持股的關鍵

## 學習方法建議

### 為什麼新手要先看小層級學習

- 小層級循環快 → 在相同時間內能觀察更多次完整的漲跌循環
- 能親眼看到：頭部、頂部、中繼整理、底部的形態
- 能感受到真實的情緒（恐懼、貪婪）→ 了解自己的心理弱點

### 學習路徑建議

```
階段 1：用極小資金在小層級（1分/5分K）做短線，練習技術和情緒控制
階段 2：技術穩定後，移到 30分K / 60分K
階段 3：有足夠資金與經驗後，以日K為主層級操作
```

> 連小層級短線都做不好，就不要談長期持股。你必須先見過頭部和底部的樣子，才能真正理解它們。

## 風控核心原則

1. **大層級決定方向**，小層級決定時機
2. **永遠要設停損**，停損位設在小層級的關鍵支撐/壓力破壞點
3. **順勢才進場**，逆勢行情觀望不操作
4. **趨勢的錨定點**：當價格突破重要支撐或阻力，代表趨勢可能轉變，需重新評估
5. **倉位管理**：大層級趨勢越清晰，可以持有越重的倉位

## 給 AI 工具的使用說明

若要用 AI（如 Claude）輔助分析 K 線，請注意以下限制：

- AI 單看小層級（如 5 分 K）**無法判斷進場點是否有效**，因為訊號品質取決於大層級脈絡
- 正確用法：先讓 AI 過濾「週K均線多排」的標的，再進入小層級找買點
- AI 適合做：**訊號過濾、條件篩選、批量掃描**，不適合做：**脫離大層級脈絡的單一時間週期判斷**
